# Vercel 绑定自定义域名步骤

## 步骤一：在 Vercel 中添加自定义域名

1. **登录 Vercel 账号**
   - 访问 [Vercel 官网](https://vercel.com/) 并登录

2. **选择你的项目**
   - 在 dashboard 中找到你已经部署的项目，点击进入项目详情页

3. **进入域名设置**
   - 点击顶部导航栏中的「Settings」（设置）
   - 在左侧菜单中选择「Domains」（域名）

4. **添加自定义域名**
   - 在「Domains」页面的输入框中，输入你购买的完整域名（例如：`www.example.com`）
   - 点击「Add」按钮

5. **获取 DNS 记录信息**
   - 添加域名后，Vercel 会显示需要设置的 DNS 记录信息
   - 通常包括两种类型的记录：
     - A 记录：指向 Vercel 的 IP 地址
     - CNAME 记录：指向 Vercel 提供的域名
   - 请记录下这些信息，稍后在阿里云域名控制台中设置

## 步骤二：在阿里云设置 DNS 记录

1. **登录阿里云控制台**
   - 访问 [阿里云官网](https://www.aliyun.com/) 并登录
   - 进入「域名控制台」

2. **找到你的域名**
   - 在域名列表中找到你要绑定的域名，点击「管理」

3. **进入 DNS 设置**
   - 在左侧菜单中选择「DNS 管理」
   - 点击「修改 DNS 解析」或类似选项

4. **添加 DNS 记录**
   - 根据 Vercel 提供的信息，添加相应的 DNS 记录：

   ### 添加 A 记录（根域名）
   - 记录类型：选择「A」
   - 主机记录：留空或输入「@」
   - 解析线路：选择「默认」
   - 记录值：输入 Vercel 提供的 IP 地址（通常是 `76.76.21.21`）
   - TTL：保持默认值

   ### 添加 CNAME 记录（www 子域名）
   - 记录类型：选择「CNAME」
   - 主机记录：输入「www」
   - 解析线路：选择「默认」
   - 记录值：输入 Vercel 提供的域名（通常是 `cname.vercel-dns.com`）
   - TTL：保持默认值

5. **保存设置**
   - 点击「保存」或「确定」按钮，保存 DNS 记录设置

## 步骤三：验证域名绑定

1. **等待 DNS 生效**
   - DNS 记录更新通常需要 5-30 分钟生效，有时可能需要更长时间

2. **在 Vercel 中验证**
   - 返回 Vercel 项目的「Domains」页面
   - 点击「Verify」按钮验证域名设置
   - 如果验证成功，域名状态会显示为「Connected」

3. **访问你的域名**
   - 在浏览器中输入你的自定义域名，检查是否能正常访问你的网站

## 步骤四：设置 HTTPS

- Vercel 会自动为绑定的域名提供免费的 HTTPS 证书
- 通常在域名验证成功后，HTTPS 会在几分钟内自动启用
- 你可以在 Vercel 的「Domains」页面查看 HTTPS 状态

## 常见问题及解决方案

### 1. DNS 记录设置后网站无法访问
- **原因**：DNS 记录可能尚未生效
- **解决方法**：等待一段时间后再尝试访问，通常需要 5-30 分钟

### 2. Vercel 验证域名失败
- **原因**：DNS 记录设置不正确或尚未生效
- **解决方法**：
  - 检查 DNS 记录是否与 Vercel 提供的信息完全一致
  - 等待 DNS 记录生效后再尝试验证

### 3. 网站可以访问但没有 HTTPS
- **原因**：HTTPS 证书正在申请中
- **解决方法**：等待 Vercel 自动为域名配置 HTTPS 证书，通常需要几分钟时间

### 4. 根域名无法访问，但 www 子域名可以访问
- **原因**：根域名的 A 记录设置不正确
- **解决方法**：检查根域名的 A 记录是否正确指向 Vercel 的 IP 地址

## 注意事项

1. **保持 DNS 记录正确**：一旦设置完成，尽量不要随意修改 DNS 记录，以免影响网站访问

2. **定期检查域名状态**：定期在 Vercel 中检查域名状态，确保绑定正常

3. **域名续费**：记得及时为域名续费，避免域名过期导致网站无法访问

4. **HTTPS 证书**：Vercel 提供的 HTTPS 证书是自动续期的，无需手动操作
